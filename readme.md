# Reisgenootjes
Onze website genaamd 'Reisgenootjes' is gemaakt voor solo reizigers 

Deze applicatie helpt solo reizigers om makkelijk reismaatjes te vinden op basis van interesses en voorkeuren.
* Kunnen registreren en inloggen;
* Interesses en voorkeuren kunnen opgeven;
* Kunnen filteren en zoeken;
* Kunnen matchen met andere reizigers;
* Kunnen chatten;

  ## Hoe bijdragen
  Om mee te kunnen helpen aan dit project kun je de onderstaande tutorial volgen:
1. "Git clone https://github.com/Nichaela/Reisgenootjes" in een lege folder
2. Maak een branch
3. npm install en alle andere dingen in terminal ("npm install nodenv", express, nodemon, ejs, mongodb, eslint, listjs)
4. Maak een ".env" file in root (project folder)
5. Plak dit in .env: "PORT=3000
DB_USERNAME=
DB_PASSWORD=
DB_HOST= blabla.blalba.mongodb.net
DB_NAME=sample_mflix
DB_COLLECTION=users"
