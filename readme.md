# Reisgenootjes
Onze website genaamd 'Reisgenootjes' is gemaakt voor solo reizigers tussen de 18 en 30 jaar oud. Deze applicatie helpt solo reizigers om makkelijk reismaatjes te vinden op basis van interesses en voorkeuren. De functies die onze website bevat zijn:
* Registreren en inloggen;
* Interesses en voorkeuren opgeven;
* Filteren en zoeken;
* Matchen met andere reizigers;
* Reizen aanmaken en delen;
* Aanmelden bij reizen van anderen;
* Chatten met andere reizigers.

## Hoe bijdragen
Om mee te kunnen helpen aan dit project kun je de onderstaande tutorial volgen:
1. "Git clone https://github.com/Nichaela/Reisgenootjes" in een lege folder;
2. Maak een branch;
3. npm install;
4. Maak een ".env" bestand in root (project folder), belangrijk: nooit dit bestand committen;
5. Plak dit in .env:
"PORT=3000
DB_USERNAME=
DB_PASSWORD=
DB_HOST= blabla.blalba.mongodb.net
DB_NAME=sample_mflix
DB_COLLECTION=users"

## Tech stack
Modules om te installeren in dit project:

- Socket.io
